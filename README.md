# 🧛 Vampire Quiz Saga

A story-driven vampire hunting quiz game built with C++ and SFML.

The player takes the role of a vampire hunter who must answer questions,
make decisions, and survive battles in a dark fantasy world.

## Features
- Story-driven gameplay
- Multiple-choice and True/False questions
- Interactive combat system
- Score tracking
- Chapter-based progression

## Technologies
- C++
- SFML
- Object-Oriented Programming

## Future Improvements
- More chapters
- More enemies
- Better animations
- Save/load system
